Configuration Supabase

Renseigner l'URL et la clé anonyme de votre projet Supabase pour activer la synchronisation entre appareils.

Instructions :
1. Créer un compte sur supabase.com (gratuit)
2. Nouveau projet → "Table Editor" → "New table" nommée orlog_blocs
3. Colonnes : id (uuid, PK), user_id (text), data (jsonb), updated_at (timestamptz)
4. Settings → API → copier l'URL et la clé "anon public"
5. Coller ici et tester
